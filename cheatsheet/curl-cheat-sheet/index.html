Curl Cheat Sheet

Written by hare on July 17, 2020 in CheatSheet, 200 words

Options

curl -o <file> # --output: write to file
curl -u user:pass # --user: Authentication
curl -v # --verbose
curl -vv # Even more verbose
curl -s # --silent
curl -i # --include: Include the HTTP-header in the output
curl -I # --head: headers only

Request

curl -X POST # --request
curl -L # follow link if page redirects

Data

curl -d 'data' # --data: HTTP post data, URL encoded (eg, status="Hello")
curl -d @file # --data via file
curl -G # --get: send -d data via get

Headers

curl -A <str> # --user-agent
curl -b name=val # --cookie
curl -b FILE # --cookie
curl -H "X-Foo: y" # --header
curl --compressed # use deflate/gzip

SSL

curl --cacert <file> --capath <dir>
curl -E, --cert <cert> # --cert: Client cert file --cert-type # der/pem/eng
curl -k, --insecure # for self-signed certs

Examples

# Post data:
curl -d password=x http://x.com/y
# Auth/data:
curl -u user:pass -d status="Hello" http://twitter.com/statuses/update.xml
# multipart file upload
curl -v -include --form key1=value1 --form upload=@localfilename URL
# Use Curl to Check if a remote resource is available
# details: https://matthewsetter.com/check-if-file-is-available-with-curl/
curl -o /dev/null --silent -Iw "%{http_code}" https://example.com/my.remote.tarball.gz
